# Bem Vindos Ao Webchat Spiral!

Este é um projeto voltado ao aprendizado, se trata de um webchat que será feito com: **Laravel 8**, **Vue.js**, **Vuex**, **Laravel WebSockets**, **Laravel Echo** e **TailwindCSS**.
