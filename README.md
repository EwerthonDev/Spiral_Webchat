# Bem Vindos Ao Webchat Spiral!

<img src="https://i.ibb.co/K5xkrGV/spiral-webchat2.png" height="350px" width="600px">

Este é um projeto voltado ao aprendizado, se trata de um webchat que será feito com: 
- **Laravel 8**
- **Vue.js**
- **Vuex**
- **Laravel WebSockets**
- **Laravel Echo**
- **TailwindCSS**
